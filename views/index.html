Getting Started

Start building your application in minutes

1. Clone & Install

Clone the repository and install dependencies to get started quickly.

git clone https://github.com/khokonm/node-express-starter-2.0.git npm install

2. Configure Environment

Set up your environment variables for database and authentication.

cp .env.example .env npm run dev

3. Start Development

Begin building your application with pre-configured routes and controllers.

npm run dev # development npm start # production
